# Talk with Buffett

**与巴菲特对话** — 不只是读信，而是坐在同一个房间里，问他你真正想问的问题。

---

## 为什么做这个？

巴菲特写了近 60 年的股东信。这些信件是价值投资最宝贵的一手资料——但大多数人的读法是错的。

脱离背景读一封信，就像脱离棋局看一步棋：你知道他走了什么，但不知道他为什么这么走。

**Talk with Buffett** 试图还原那个语境：让你在 1999 年网络泡沫的背景下，问他为什么没有买科技股；让你在 2008 年金融危机之后，问他那时候在想什么。

---

## 核心功能

### 与虚拟巴菲特对话

输入任何关于投资的问题，虚拟巴菲特会用他自己的话回答——每个回答都基于真实信件，每句话都有来源可查。

```
你：护城河这个概念，你怎么理解？

巴菲特：我一直有个规则：如果你不能用一句话说清楚这家公司为什么
        比竞争对手强，那护城河可能根本不存在……

        📎 来源：1995 年股东信
```

### 双语对照阅读

全部信件中英文并排展示，左侧英文原文、右侧中文译文，滚动自动联动。读到任何一段，随时可以向巴菲特提问。

### 引用可溯源

每一条回答都标注具体来源（年份 · 段落），一键跳转原文。不是 AI 编造，是他真实说过的话。

### 数字人模式 *(即将上线)*

视频 + 声音，让对话从文字变成面对面。
