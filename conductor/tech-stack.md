# Tech Stack: Learn from Buffett

## Core Framework
- **Next.js (App Router, TypeScript):** Used for both frontend and backend (API routes).
- **React:** For user interface components.

## Data Persistence & Access
- **Prisma ORM:** For type-safe database access and migrations.
- **SQLite:** Used for data persistence (initial development/production).

## Authentication & Authorization
- **NextAuth.js:** For user authentication and session management.

## Styling & UI
- **Vanilla CSS / CSS Modules:** Used for consistent, performant styling across the application.

## Data Processing & Utilities
- **Python:** For PDF crawling, parsing, and letter segmentation.
- **ts-node:** For running TypeScript scripts in development.

## AI Integration
- **OpenAI GPT-4 API:** For deep analysis of shareholder letters and financial concepts.

## Testing Framework
- **Vitest:** For unit and integration testing with >80% coverage target.
